## tendermint-node-cli

#### How to use?

```bash

Mini-Manual

1. Confiquire your query using subscribe [query_name] command (For now, you can only subscribe to one endpoint at a time).

2. Send subscrition request to your node using init command.  

3. To unsubscribe, use stop command.

4. To see list of available queries, use subscribe command.

5. To quit app, use exit comman or ctrl+C

```
